# Crypto Spread Scanner

Web-дашборд для отслеживания арбитражных позиций (Short vs Long) на разных криптовалютных биржах.

## Возможности
- Расчет Spread PnL, Total PnL, PnL @ Close в реальном времени.
- Автоматический сбор текущих цен по REST API (Binance, Hyperliquid, Bitget, Gate.io).
- Удобный темный интерфейс.

## Инструкция по запуску
Так как проект написан на чистом HTML/CSS/JS, для его использования не нужны локальные сервера, Node.js или React.

1. Откройте папку `d:\Crypto_Scanner` в проводнике.
2. Просто дважды кликните по файлу `index.html`, чтобы открыть его в любом браузере (Chrome, Safari, Edge).
3. Добавьте тикер в поле "Ticker" (например, `BTC` или `BTCUSDT`).
4. Введите ваши цены входа (Entry) и объем (Amount). Текущие цены подтянутся автоматически (обновление каждые 5 секунд).
